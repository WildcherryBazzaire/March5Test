What are Wireframes?

Wireframes speaking are constructed concepts of how your page will look like. This allows visualization on how your page will intend to look like and where the elements will be layed out. We use wire frames to build are visualized website and to work on the view of it with other teammates so everyone is on the same page on what their intended to do. So essentially, wireframes give a visual on the final product/webpage will intend to look

Git Workflow

Gitflow works as a collaboration situation where a team works on code of the same project together with the use of git operations such as merging,branchingm and pull request. With merging with gitflow, this allows teams to merge updated code with their code and to work out the changes between them. With branches, this allows a team to work on seperate changes to the project code without affecting the master/working code, or better yet to implement ideas or to fix bugs. With pull request, a branch is set up for review for to merge with the master if the implenmented idea works.
